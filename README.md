# ITI HA ASA Foods - Cafe Menu Website

A modern, responsive cafe menu website for ITI HA ASA Foods, featuring Fresh Bean and Bean & Brew brands. Built with React and enhanced with beautiful animations using Framer Motion.

## 🌟 Features

- **Fully Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI/UX** - Clean, attractive design with smooth animations
- **Interactive Menu** - Browse through different categories (Coffee, Tea, Snacks, Mocktails)
- **Animated Components** - Engaging user experience with Framer Motion animations
- **Frontend Only** - Pure client-side application, no backend required
- **Performance Optimized** - Fast loading and smooth interactions

## 🎨 Design Highlights

- **Color Scheme**: Warm cafe-inspired colors (#3E2723, #FFF8E1, #D7CCC8)
- **Typography**: Playfair Display for headings, Lato for body text
- **Animations**: Subtle, professional animations that enhance user experience
- **Sections**: 
  - Hero section with brand identity
  - Interactive menu with category filtering
  - About section showcasing cafe story and values
  - Footer with social links and contact information

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Akhi3011/ITI-HA-ASA-Foods.git
cd ITI-HA-ASA-Foods
```

2. Navigate to the frontend directory:
```bash
cd frontend
```

3. Install dependencies:
```bash
npm install
# or
yarn install
```

4. Start the development server:
```bash
npm start
# or
yarn start
```

5. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## 📦 Build for Production

```bash
npm run build
# or
yarn build
```

This creates an optimized production build in the `build` folder.

## 🛠️ Technologies Used
